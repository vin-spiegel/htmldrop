# Contributing

Thanks for your interest! htmldrop is maintained on a best-effort basis.

- **Bug reports**: open an issue with reproduction steps. Include the endpoint
  or MCP call you made and what you got back.
- **Security issues**: do **not** open a public issue — see [SECURITY.md](SECURITY.md).
- **Pull requests**: welcome for bug fixes and small improvements. For larger
  features, open an issue first so we can discuss direction — the project
  intentionally stays small ("just the essentials, nothing to configure").
- **Dev setup**: `pnpm install && pnpm dev`, tests with `pnpm test`. CI runs
  build + tests on every PR.
